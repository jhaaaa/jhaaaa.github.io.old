About Haiku Mornings

👋 I’m J-Ha. That’s 90s-speak for Jennifer Hasegawa.

I’m a UX writer, technical writer, and information architect.

At a previous job, I learned to use git to maintain technical documentation. It was a love/hate relationship.

Now that my current job doesn’t involve much git, I found myself missing it.

I don’t have any gitfu moves, but a lot of 🩸, 😅, and 😭 went into learning the few moves I know. I created this site as a way to keep my git fresh. This site is built using GitHub Pages and Jekyll and I publish posts using git.

✍️✍️✍️

Oh, I’m also a poet, photographer, and performance artist. You can learn more about my work @ J-Ha Land.
